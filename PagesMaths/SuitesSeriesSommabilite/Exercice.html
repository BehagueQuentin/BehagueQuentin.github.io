Exercice 1 :

Quelle est la nature de la série de terme général : \[u_n = \cos{\left(n^2\pi \ln{\left(1-\frac{1}{n}\right)}\right)}\]

...

Soit \(n\in \mathbb{N}^* :\) \[ u_n =\cos{\left(n^2\pi \ln{\left(1-\frac{1}{n}\right)}\right)} =\cos{\left(n^2\pi \left( -\frac{1}{n} - \frac{1}{2n^2}-\frac{1}{3n^4} + O\left(\frac{1}{n^3}\right)\right)\right)}\] \[ = \cos{\left( -\pi n- \frac{\pi}{2}-\frac{\pi}{3n} + O\left(\frac{1}{n^2}\right)\right)} = \sin{\left( \frac{\pi}{3n} + O\left(\frac{1}{n^2}\right)\right)} \]\[ = (-1)^n \frac{\pi}{3n} + O\left(\frac{1}{n^2}\right)\] On conclut avec le critère de convergence des séries alternées. Ainsi : \[\boxed{\sum \cos{\left(n^2\pi \ln{\left(1-\frac{1}{n}\right)}\right)}\textrm{ est une série convergente}}\]

Exercice 3 :

Montrer la convergence et calculer la somme de la série : \[\sum_{n = 1}^N \ln\left(\frac{(2n+1)n} {(2n-1)(n+1)}\right) \]

...

Soit \(N\) un entier naturel : \[S_N = \sum_{n = 1}^N \ln\left(\frac{(2n+1)n} {(2n-1)(n+1)}\right) = \ln \left(\prod_{n = 1}^N\frac{(2n+1)n} {(2n-1)(n+1)}\right) = \ln \left(\prod_{n = 1}^N\frac{(2n+1)} {(2n-1)}\prod_{n = 1}^N\frac{n} {(n+1)}\right) \] \[= \ln \left(\frac{2N+1} {N+1} \right) \longrightarrow \ln(2)\] Ainsi, on obtient le résultat voulu : \[\boxed{\sum \ln\left(\frac{(2n+1)n} {(2n-1)(n+1)}\right) \textrm{converge vers } \ln(2)}\]

Exercice 5 :

  1. Donner une condition nécessaire sur la suite \((u_n)\) pour que la série numérique \(\sum u_n\) converge.
  2. Cette condition est-elle suffisante ? Justifier.
  3. Déterminer la nature de la série de terme général \(v_n = \sqrt{n} + a\sqrt{n+1} + b\sqrt{n+2}\) en fonction des réels \(a\) et \(b\).

...

  1. Une condition nécessaire est que la suite tend vers 0 lorsque n tend vers l'infini. On le montre par contraposée, si la suite \((S_n)\) converge, alors (sans détailler) : \[|S_n - S_{n+1}| = |S_n - l + l - S_{n+1}| \leq |S_n - l| + |S_{n+1} - l|\leq 2\epsilon\] Ainsi : \[\boxed{\sum S_n \textrm{ converge} \Longrightarrow S_{n+1} - S_n = u_{n+1} \longrightarrow 0 }\]
  2. Cette condition n'est pas suffisante, il suffit pour voir cela de prendre la suite harmonique, qui diverge. En effet : \[ H_{2n} - H_n = \sum_{k = n+1}^{2n} \frac{1}{n}\leq \sum_{k = n+1}^{2n}\frac{1}{2n} = \frac{1}{2} \ne 0 \] Ainsi, on exhibe un contre exemple et : \[\boxed{u_n \longrightarrow 0 \textrm{ n'implique pas } \sum u_n \textrm{ converge}}\]
  3. Très simplement, on effectue un DL : \[ v_n = \sqrt{n} + a\sqrt{n+1} + b\sqrt{n+2} = \sqrt{n}\left( 1 + a\sqrt{1 + \frac{1}{n}} + b\sqrt{1 + \frac{2}{n}}\right)\] \[ =\sqrt{n}\left( 1+ a\left( 1 + \frac{1}{2n} - \frac{1}{8n^2} + \frac{1}{16n^2} + O\left(\frac{1} {n^4}\right)\right) + b\left( 1+ \frac{1}{n} - \frac{1}{4n^2} + \frac{1}{2n^3} + O\left(\frac{1} {n^4}\right)\right) \right)\] \[ = \sqrt{n}\left( 1+ (a+b) + \frac{a+2b}{2n} - \frac{a+2b}{8n^2} + \frac{a+8b}{16n^3} + O\left(\frac{1} {n^4}\right) \right) \] Ainsi, pour garantir la convergence, on doit avoir : \[\left\{ \begin{array}{ll} a + b = -1\\ a+2b = 0 \end{array} \right. \Longrightarrow \left\{ \begin{array}{ll} a = -2 \\ b = 1 \end{array} \right.\] Enfin, on obtient le résultat : \[\boxed{ \sum v_n \textrm{ converge } \Longleftrightarrow \left\{ \begin{array}{ll} a = -2 \\ b = 1 \end{array} \right.}\]

Exercice 9 :

On définit la série harmonique par \[\forall n\in\mathbb{N}^*, \ H_n = \sum_{k=1}^n \dfrac{1}{k}.\]
  1. La suite \((H_n)_{n\in\mathbb{N}^*}\) converge-t-elle ?
  2. Montrer que \(H_n \underset{n\to+\infty}\sim \ln(n)\).
    On pose la suite \((u_n)_{n\in\mathbb{N}^*}\) définie par \(\forall n\in\mathbb{N}^*, \ u_n = H_n-\ln(n).\)
  3. Montrer que \((u_n)_{n\in\mathbb{N}^*}\) converge vers un réel \(\gamma\).
  4. On pose \(w_n = u_n-\gamma\). Déterminer un équivalent de \(w_n\).

...

  1. La suite ne converge pas, on peut pour cela montrer que la différence \(H_{2n} - H_n\) est toujours supérieure à \(0.5\) or, si la série convergeait, cette différence devrait tendre vers 0.
  2. On a pour tout \(k \in \mathbb{N}\) et pour tout \(t\in[k; k+1]\) l'inégalité : \[\int_{k}^{k+1} \frac{dt}{t}\leq \frac1k\leq \int_{k-1}^k \frac{dt}t\] On somme ces relations, puis on ajoute le terme 1 manquant : \[1+\int_{2}^{n+1}\frac{dt}{t}\leq H_n\leq 1+\int_1^n \frac{dt}{t}.\] On trouve alors : \[1+\ln(n+1)-\ln(2)\leq H_n\leq \ln(n)+1.\] De cet encadrement, on déduit l'équivalence : \[\boxed{H_n\sim \ln n}\]
  3. Des relations écrites à la question précédente, on déduit que : \[H_n - \ln(n) \geq H_n - \ln(n+1)\geq 1-\ln(2)\] Or : \[H_{n+1} - \ln(n+1) - H_n + \ln(n) = \frac{1}{n+1} - \ln\left(1+\frac{1}{n}\right)\sim \frac{1}{n+1} -\frac{1}{n} \] Donc la suite est décroissante et minorée par \(1-ln(2)\) donc est convergente.
  4. On trouve facilement, avec un développement limité que : \[t_n=w_{n+1}-w_n=-\frac{1}{2(n+1)^2}+o\left(\frac1{n^2}\right)\] La série de terme général \(t_n\) est convergente. On a l'équivalent suivant des restes : \[\sum_{k=n}^{+\infty}\big(w_{k+1}-w_k\big)\sim_{+\infty}\sum_{k=n}^{+\infty}\frac{-1}{2k^2}.\] Soit \(N\in \mathbb{N}\), on sait que \((w_n)\) tend vers \(0\), alors : \[\sum_{k=n}^N\big( w_{k+1}-w_k\big)=w_{N+1}-w_n\to -w_n\] D'où : \[w_n \sim \sum_{k=n}^{+\infty}\frac{1}{2k^2}\sim \frac{1}{2n}\] (Cette dernière équivalence se montre par comparaison série intégrale.) Enfin : \[\boxed{H_n=\ln n+\gamma+ \frac{1}{2n} + o\left(\frac{1}{n^2}\right)}\]

Exercice 11 :

Soit \((u_n)\) une suite. On pose \(\displaystyle v_n = \frac{u_1+u_2+...+u_n}{n}\).
  • Montrer que si la suite \((u_n)\) est monotone, il en va de même de la suite \((v_n)\).
  • Montrer que si \((u_n)\) converge, \((v_n)\) converge vers la même limite. (Théorème de Césaro)
  • Procéder de même avec \(\displaystyle v_n = \frac{1}{2n}\sum_{k = 0}^{n}\binom{n}{k}u_n\).

...

  1. En mettant au même dénominateur, le numérateur vaut : \[v_{n+1} - v{n} = u_1+u_2+...+u_n - nu_{n+1} = (u_1 - u_{n+1}) + ... + (u_n - u_{n+1})\] On conclut que ces suites ont même monotonie.
  2. Soit \(N_1 \in \mathbb{N}\) tel que pour tout \(n \geq N_1, |u_n -l| \leq \varepsilon\). Alors : \[|v_n - l| = \left| \frac{1}{n} \sum^{n}_{k = 1} u_n -l \right| = \left| \frac{1}{n} \sum^{n}_{k = 1} (u_n -l) \right| \leq \frac{1}{n} \sum^{n}_{k = 1} |u_n -l| \] \[|v_n - l| \leq \frac{1}{n}\sum^{N_1}_{k = 1} (u_n -l) + \frac{1}{n}\sum^{n}_{k = N_1 + 1} \varepsilon \] Or la première somme est un terme constant, donc, à partir d'un certain rang \(N_2 \geq N_1\) on aura \[|v_n - l| \leq 2\varepsilon\] Donc : \[\boxed{(v_n) \textrm{ tend vers } l} \]
  3. On procède en notant \(u_n = l + \varepsilon(n)\) avec \(\varepsilon(n) \longrightarrow 0\), on en déduit alors une nouvelle forme de l'égalité, et on procède comme précédemment.

Exercice 12 :

Soit \({{\left( {{u}_{n}} \right)}_{n\in \aleph }}\) et \({{\left( {{v}_{n}} \right)}_{n\in \aleph }}\) deux suites, \({{\left( {{v}_{n}} \right)}_{n\in \aleph }}\) étant décroissante vers 0. On considère \({{S}_{n}}=\sum\limits_{k=0}^{n}{{{u}_{k}}.{{v}_{k}}}\). On pose \({{U}_{n}}=\sum\limits_{k=0}^{n}{{{u}_{k}}}\).
  1. Exprimer \({{S}_{n}}\) sous forme d'une somme faisant intervenir \({{U}_{k}}\) et \(\left( {{v}_{k+1}}-{{v}_{k}} \right)\).
  2. En déduire que si les termes \({{U}_{k}}\) sont bornés et que si la suite \({{\left( {{v}_{n}} \right)}_{n\in \aleph }}\) converge, la série \({{S}_{n}}=\sum\limits_{k=0}^{n}{{{u}_{k}}.{{v}_{k}}}\) converge.
  3. Appliquer cela pour montrer que \(\sum{\frac{\sin (n\theta)}{n}}\) converge.

...

  1. On exprime chaque \(u_k\) comme une différence pour pouvoir séparer la somme puis ré-indexer. On fera attention à retirer certains termes pour lesquels cette transformation ne s'applique pas :\[ \sum_{k=0}^n u_kv_k = \sum_{k=1}^q (U_{k}-U_{k-1})v_k + u_0v_0 = \sum_{k=1}^n U_{k}v_k-\sum_{k=1}^n U_{k-1} v_k + u_0v_0 \] \[=\sum_{k=1}^{n} U_kv_{k}-\sum_{k=0}^{n-1} U_k v_{k+1} + u_0v_0 =\sum_{k=1}^{n-1} U_k(v_k-v_{k+1})+ U_nv_n - U_{0}v_{1} + u_0v_0\] \[=\boxed{\sum_{k=0}^{n-1} U_k(v_k-v_{k+1})+ U_nv_n }\]
  2. On se sert de l'écriture précédente, soit \(\varepsilon>0\) tel que \(|v_n| \leq \varepsilon\) et soit \(M\in \mathbb{R}\) tel que pour tout \(k, |u_k| \leq M\) : \[\left|\sum_{k=0}^n u_kv_k\right|\leq M\varepsilon +\left|\sum_{k=0}^{n-1}M(v_k-v_{k+1})\right|\] Or, par critère de comparaison suite série, la série de droite converge, le premier terme du membre de droite tend vers 0. On en déduit que : \[\boxed{\sum u_kv_k \textrm{ est une série convergente. }}\]
  3. D'après la question précédente,il suffit de montrer que la suite : \[S_n=\sum_{k=0}^n \sin(k\theta)\] est bornée. Pour cela il suffit de remarquer que : \[\sum_{k=0}^n \sin(k\theta)=\Im m\left(\sum_{k=0}^n e^{ik\theta}\right)\] Puis, en prenant \(\theta\notin2\pi\mathbb Z\), il vient : \begin{eqnarray*} \sum_{k=0}^n e^{ik\theta}&=&\frac{1-e^{i(n+1)\theta}}{1-e^{i\theta}}
    &=&\frac{e^{i\frac{(n+1)}{2}\theta}}{e^{i\theta/2}}\times\frac{e^{-i(n+1)\theta/2}-e^{+i(n+1)\theta/2}}{e^{-i\theta/2}-e^{i\theta/2}}
    &=&e^{in\theta/2}\frac{\sin\left(\frac{(n+1)\theta}{2}\right)}{\sin\left(\frac{\theta}{2}\right)} \end{eqnarray*} Enfin on a : \[S_n=\frac{\sin\left(\frac{n\theta}{2}\right)\sin\left(\frac{(n+1)\theta}{2}\right)}{\sin\left(\frac{\theta}{2}\right)} \leq \frac{1}{|\sin(\theta/2)|}\] D'après la question précédente : \[\boxed{\sum{\frac{\sin (n\theta )}{n}} \textrm{ converge.}}\]
